منصة بيع وشراء قطع غيار السيارات في الجزائر.

## تشغيل المشروع

```bash
npm install
npm run init-admin   # أنشئ حساب المدير
npm run seed         # أضف 25 قطعة غيار حقيقية بصور وأسعار DZD
npm run dev          # شغّل الخادم على المنفذ 3000
```
