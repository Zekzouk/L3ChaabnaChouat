# SAYARATI - Car Parts Marketplace 

##  Project Description

SAYARATI is a full-stack web application for buying and selling car parts. It allows users to register, log in, browse products, place orders, and manage dashboards for buyers, sellers, and administrators.


##  Tech Stack

* Frontend: HTML, CSS, JavaScript
* Backend: Node.js, Express.js
* Database: MongoDB with Mongoose


##  Features

* User registration and login system
* Role-based dashboards (Buyer / Seller / Admin)
* Product listing and details page
* Shopping cart and checkout system
* Order management
* Admin control panel

---

## 📁 Project Structure

* `/public` → Frontend pages (HTML/CSS/JS)
* `/routes` → API endpoints
* `/models` → Database schemas
* `/middleware` → Authentication logic
* `/server.js` → Main backend entry point

---

##  How to Run the Project

```bash
npm install
node server.js
```

Then open:


http://localhost:3000



  Database Setup

Make sure you create a `.env` file:

```
MONGODB_URI=your_mongodb_connection_string
PORT=3000
```
